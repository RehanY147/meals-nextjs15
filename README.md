# ChaiTea Food - A NextJs project
This is a personal project to explore some new ideas to implement a simple full stack app with some CRUD operations using NextJs tools. Forms with server actions, image pickers, file uploads, validation, directory/file system based routing and other concepts to stay strictly within the NextJs ecosystem.
